## About

This repository contains solutions to **52 Python programming problems** assigned as part of the **CCE-3 (Continuous Course Evaluation 3)** for the subject **Programming for Problem Solving (PPS)**. Each program reinforces core Python concepts including:

- **Data Structures** — Lists, Tuples, Dictionaries
- **String Manipulation** — Searching, replacing, reversing, counting
- **Numerical Operations** — Averages, totals, comparisons, sorting
- **Control Flow** — Loops, conditionals, nested logic
- **Collections** — Frequency counting, duplicate detection, filtering

## Problem Statements

### Programs 01 – 10

**01 — Highest Sale**  
Given a list of daily sales figures, find and display the highest sale value.

**02 — Count Tuple Elements**  
Given a tuple of elements, count and display the total number of elements in it.

**03 — Lowest Temperature**  
Given a list of temperature readings, find and display the lowest recorded temperature.

**04 — Search Tuple**  
Given a tuple and a target value, search for the target and display whether it is found or not.

**05 — Total Marks**  
Given a list of student marks, compute and display the total marks.

**06 — Stock Greater Than 50**  
Given a list of stock quantities, display all stocks whose quantity is greater than 50.

**07 — Find Topper**  
Given a dictionary of student names and their marks, find and display the student with the highest marks.

**08 — Even Product IDs**  
Given a list of product IDs, display only the IDs that are even numbers.

**09 — Pass Students Count**  
Given a list of student marks, count and display how many students passed (marks >= 40).

**10 — Reverse Queue**  
Given a list representing a queue of elements, reverse the queue and display the result.

### Programs 11 – 20

**11 — Word Count**  
Given a sentence string, count and display the total number of words in it.

**12 — Duplicate Orders**

**13 — Replace Word**  
Given a sentence and two words, replace every occurrence of the first word with the second word and display the result.

**14 — Students Above 50**  
Given a list of student marks, display the marks of all students who scored above 50.

**15 — Remove Spaces**  
Given a string, remove all leading, trailing, and extra spaces and display the cleaned string.

**16 — Employee ID Search**  
Given a dictionary of employee IDs and names, search for a given employee ID and display the corresponding name if found.

**17 — Highest Marks N Students**  
Given a list of student marks, find and display the top N highest marks.

**18 — Customer Count**  
Given a dictionary of customer categories and counts, display the total number of customers across all categories.

**19 — Average Temperature**  
Given a list of temperature readings, calculate and display the average temperature.

**20 — Highest Price Tuple**  
Given a tuple of item prices, find and display the highest price.
